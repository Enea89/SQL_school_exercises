# Esercizi SQL - Scuola
Raccolta di laboratori di database svolti in classe.

## Esercizio: Articoli Fatture
Questo esercizio si focalizza sul creare un database di articoli con delle fatture e di gestione di dati base.

### Obiettivi raggiunti:
- **Integrità dei dati**: Implementazione di `PRIMARY KEY`, `FOREIGN KEY` e vincoli.
- **Analisi dati**: Scrittura di query complesse con `JOIN` multiple per collegare Clienti, Fatture e Articoli.
- **Logica temporale**: Filtri avanzati sulle date per estrarre report degli ultimi anni.

### Consultare il codice:
Il file `esercizio_fatture.sql` contiene sia la parte di creazione tabelle che le query risolte.
